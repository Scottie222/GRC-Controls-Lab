# ISO 27001 – Vendor Risk Management

## Overview
Third-party vendors can introduce risks. Capital One used cloud services that contributed to misconfigurations.

## Controls
- ISO 27001: A.15.1 & A.15.2  
- NIST CSF: ID.SC-1  
- GDPR: Art. 28  
- POPIA: Sec. 19–21

## Implementation
Simulated vendor audits via Python scripts and GitHub Actions.
