# ISO 27001 – Incident Response

## Overview
Capital One failed to detect incidents quickly. Proper monitoring and response reduces impact.

## Controls
- ISO 27001: A.16.1  
- NIST CSF: DE.CM-1 to DE.DP-4  
- GDPR: Art. 33–34  
- POPIA: Sec. 22–23

## Implementation
Python script to generate incident logs, mini dashboards, GitHub Actions alerts.
